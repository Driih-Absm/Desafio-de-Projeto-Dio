Comandos do Git 

No cmd, o comando dir - serve para listar todas as pastas

Comando cd, possibilita navegar entre as pastas, então é só colocar cd e o nome da pasta

Comando cd .. , possibilita voltar para a pasta anterior.

Comando cls, limpa a tela.

Comando mkdir, cria uma pasta.

Comando echo serve para devolver uma frase, se eu digitar echo hello > hello.txt ele cria um arquivo.

Comando del e o nome da pasta, deleta todos os arquivos de dentro da pasta.

Comando rmdir e o nome da pasta e o /S /Q deleta a pasta inteira.

SHA1 serve para criar uma encriptação de cada alteração. É gerado um código de 40 dígitos.

O comando a ser utilizado é openssl sha1 e o nome do arquivo, e pronto é gerado um código com 40 dígitos. Isso serve para ajudar na identificação, pois quando uma vírgula é alterada o código também é alterado e se retirarmos a vírgula e gerarmos outro código, ele retorna para o primeiro código.
